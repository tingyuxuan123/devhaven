## ADDED Requirements
### Requirement: 开发工具维护
系统 SHALL 允许用户维护开发工具列表（含内置预设与自定义），并在设置中启用/禁用工具。

#### Scenario: 用户添加自定义工具
- **WHEN** 用户在设置中新增一个自定义工具并保存
- **THEN** 该工具应出现在项目卡片的工具选择列表中

### Requirement: 预设检测与展示
系统 SHALL 仅展示已检测到安装的内置预设工具。

#### Scenario: 预设未安装
- **WHEN** 系统检测到某预设工具未安装
- **THEN** 该预设不应出现在工具选择列表中

### Requirement: 默认开发工具
系统 SHALL 支持设置默认开发工具，并在“一键打开项目”时使用该工具。

#### Scenario: 使用默认工具打开
- **WHEN** 用户点击项目卡片的默认打开入口
- **THEN** 系统使用默认开发工具打开该项目路径

### Requirement: 项目卡片工具选择
系统 SHALL 在项目卡片提供“开发工具”选择入口，允许用户选择具体工具打开项目。

#### Scenario: 选择工具打开项目
- **WHEN** 用户在项目卡片选择某个开发工具
- **THEN** 系统使用该工具打开对应项目路径
